## Build Instructions

### Prerequisites

- Java JDK 8 or later
- Apache Maven 3.x or later

### Compile the Project

```bash
mvn clean compile
```

### Run the Application

```bash
mvn exec:java
```

Or, if an executable JAR is generated:

```bash
java -jar target/<jar-file-name>.jar
```
